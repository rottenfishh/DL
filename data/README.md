# data

В этой директории должны содержаться parquet файлы датасетов контеста

## Содержимое

### Исходные файлы
- `train_target.parquet` — таргеты для train.
- `sample_submit.parquet` — пример файла сабмита.
- `train_main_features.parquet` — основной набор train-признаков.
- `train_extra_features.parquet` — дополнительный набор train-признаков.
- `test_main_features.parquet` — основной набор test-признаков.
- `test_extra_features.parquet` — дополнительный набор test-признаков.
